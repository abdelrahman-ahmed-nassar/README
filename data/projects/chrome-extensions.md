# Project: Chrome Extensions

**Type:** Browser Extensions
**Category:** Utility / Productivity Tools
**Status:** ✅ Published (open source)

---

## Overview

Abdelrahman built several small Chrome extensions to solve specific personal frustrations with social media and productivity. These are lightweight utility tools, not major projects, but demonstrate his ability to ship focused solutions quickly.

---

## Extensions

### 1. Facebook Feed Eradicator

**Problem:** Facebook's news feed is distracting and addictive.
**Solution:** A Chrome extension that removes the Facebook news feed entirely, allowing you to use Facebook for messaging and groups without the distraction of the feed.

**Repo:** https://github.com/abdelrahman-ahmed-nassar/facebook-feed-eradicator-chrome-extensions

---

### 2. Unlike YouTube Videos

**Problem:** Cleaning up liked YouTube videos is a slow, manual, one-by-one process.
**Solution:** A Chrome extension that automates bulk un-liking of YouTube videos with a single click.

**Repo:** https://github.com/abdelrahman-ahmed-nassar/unlike-youtube-videos-chrome-extensions

---

### 3. LinkedIn Accept Connections

**Problem:** Accepting many LinkedIn connection requests manually is tedious.
**Solution:** A Chrome extension that automates bulk-accepting LinkedIn connection requests.

**Repo:** https://github.com/abdelrahman-ahmed-nassar/LinkedIn-accept-connections-extensions

---

## Tech Stack

- JavaScript (vanilla)
- Chrome Extension APIs (Manifest V3)
- DOM manipulation

---

## The Pattern

All three extensions follow the same philosophy as QuickTube: identify a specific personal pain point → build the simplest possible automation → ship it. They're small but real tools that solve real problems.
