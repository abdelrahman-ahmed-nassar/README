# Project: Al-Azhar Electronics Showcase Platform

**Type:** Full-Stack Web Application
**Category:** Official University Platform
**Status:** ✅ Production — Live & Active
**URL:** https://al-azhar-electronics.netlify.app/
**Built:** First year at Computer Engineering (2024)
**Supervisor:** Prof. Khairy

---

## The Problem

Student hardware projects at the Computer & Systems Engineering department at Al-Azhar University had no permanent home. After each academic year ended, projects were simply discarded — there was no centralized, accessible archive to preserve them, showcase student work, or create lasting portfolios.

---

## The Solution

Instead of submitting a hardware project for his electronics course, Abdelrahman proposed and built a full-stack web platform where students can upload, document, and permanently showcase their engineering projects.

Each student gets an auto-generated portfolio. Each team gets a dedicated profile page. The faculty gets a professional window into all student work, while external visitors can browse projects and connect with student teams.

---

## Tech Stack

| Layer            | Technology                        |
| ---------------- | --------------------------------- |
| Frontend         | Next.js, TypeScript, Tailwind CSS |
| Backend/Database | Supabase, PostgreSQL              |
| Storage          | Cloudinary (image uploads)        |

---

## Key Features

- **Project Browsing & Search** — Search by team name, student name, or project keyword
- **Team Profiles** — Dedicated pages showing specialties, members, and all their projects
- **Full Project Pages** — Goals, technologies used, implementation details, videos, documents, and contributors
- **Auto-Generated Student Portfolios** — Every student gets a portfolio page built from their submitted projects
- **Student Dashboard** — Login system with CRUD operations: update profile, create teams, submit projects
- **Futuristic UI** — Design inspired by electronics and circuit aesthetics
- **Scalable Architecture** — Built to handle the entire department's project history over many years
