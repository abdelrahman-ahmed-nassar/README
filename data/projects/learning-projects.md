# Learning Projects — Abdelrahman Nassar

These are projects built while learning web development through courses. They are not client or startup projects, but several are polished enough to be portfolio pieces. Total: **20+ projects** across the learning journey.

---

## Highlighted Projects

### Forkify (JavaScript)

A recipe finder web application with access to over 1,000,000 recipes.

- Search recipes by ingredient or name
- Adjust serving sizes with automatic ingredient scaling
- Bookmark and save favorite recipes
- Built with vanilla JavaScript, MVC architecture

**Live:** https://abdelrahman-forkify.netlify.app/

---

### Omnifood (HTML + CSS)

A modern, fully responsive restaurant/food subscription landing page.

- Smooth scroll animations
- Pricing section, testimonials, features showcase
- Fully functional contact form
- Optimized for all screen sizes
- Built with pure HTML5 and CSS3 (Flexbox + Grid)

**Live:** https://abdelrahman-ahmed-nassar.github.io/omnifood/

---

### Bankist App (JavaScript)

A banking application simulation.

- Account login system
- Money transfers between accounts
- Loan requests
- Transaction history with dates
- Account closure
- Auto logout timer
- Built with vanilla JavaScript

**Live:** https://abdelrahman-ahmed-nassar.github.io/bankist/

---

### Mapty (JavaScript)

A workout tracking app with map integration.

- Log running and cycling workouts
- Set distance, duration, cadence/elevation
- Workouts pinned on an interactive map (Leaflet.js)
- Data persisted in localStorage
- Built with vanilla JavaScript and OOP principles

**Live:** https://abdelrahman-ahmed-nassar.github.io/mapty/
