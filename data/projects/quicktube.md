## The Problem

Downloading YouTube videos and playlists is frustratingly complex:
- Existing tools require command-line knowledge
- Libraries like `yt-dlp` and `ffmpeg` need manual installation and configuration
- Non-technical users are left out entirely

---

## The Solution

QuickTube is a simple, fast, and user-friendly Python desktop app that wraps `yt-dlp` and `ffmpeg` into a clean experience — packaged as a single `.exe` file.

**Users download it, open it, and immediately start downloading videos — no setup, no configuration, no command line.**

---

## Key Features

- Download individual YouTube videos
- Download entire YouTube playlists
- Simple and intuitive UI (no technical knowledge required)
- Packaged as a standalone `.exe` — zero installation needed
- Fast download speeds using yt-dlp under the hood
- Audio extraction supported
- Quality selection options
